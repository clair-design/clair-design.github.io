日期限制

通过value可以传入默认显示的日期，minDate和maxDate可以对日期的选择进行限制， pattern指定返回的日期格式

​
​2017-09​​
日一二三四五六
272829303112
3456789
10111213141516
17181920212223
24252627282930
1234567
​ ​
<c-calendar
  :value="'2017-9-15'"
  :maxDate="'2018-8-15'"
  :minDate="'2017-8-15'"
  @update="dateChange"
  :pattern="'yyyy/MM/dd'"
>
</c-calendar>

<script>
export default {
  methods: {
    dateChange (date) {
      alert('您选择的日期是：' + date)
    }
  }
}
</script>

月份

通过type可以设置显示月份选择

​
​2017-09​​
一月二月三月
四月五月六月
七月八月九月
十月十一月十二月
​ ​
<c-calendar
  :type="'month'"
  :value="'2017-9'"
  :maxDate="'2018-8-15'"
  :minDate="'2017-8-15'"
  @update="dateChange"
  :pattern="'yyyy/MM'"
>
</c-calendar>

<script>
export default {
  methods: {
    dateChange (date) {
      alert('您选择的日期是：' + date)
    }
  }
}
</script>
